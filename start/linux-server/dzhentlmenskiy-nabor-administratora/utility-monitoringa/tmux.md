`htop` -- Утилита для мониторинга ресурсов и управления процессами

## Установка:

-  Debian/Ubuntu/Astra - `sudo apt install htop`

-  Fedora/RedHat/RedOS - `sudo dnf install htop`

## Использование:

Для запуска используется команда `htop`
