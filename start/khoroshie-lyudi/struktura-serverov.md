**Сервер 192.168.100.2**

**Сервер 192.168.100.3**

Сервер предназначен чтобы работала ТЕЛЕФОНИЯ. На данном компьютере располагается СУБД MS SQL с базой телефонии и ПО для телефонии Panasonic + 1С Рарус.

ПО : PostgreSQL, MS SQL, Panasonic, 1С-Рарус

**Сервер 192.168.100.10**
